# flag-memory-experiment

本プロジェクトは、大学修士課程（M1）の研究のために開発しましたが、  
**研究計画の変更により、実際の実験では使用されていません。**

PC（Web ブラウザ）上で実施する実験システムとして設計されており、  
被験者は Web ブラウザ上で課題を行い、課題中には一定間隔で時刻情報が表示される想定でした。

本システムは、  
**国旗（画像）をめくりながら記憶していく課題**と  
**時間提示を組み合わせた実験**を想定して開発されています。

## 構成
- `client/` : 被験者向け Web UI（記憶課題・時刻表示など）
- `server/` : PHP バックエンド  
  （※本リポジトリには **サンプル実装のみ** を含み、実運用コードやデータは含まれていません）
